尼姆游戏规则

第 1 章

尼姆（Nim）是最经典的"取石子游戏"：

1. 有几堆石子，每堆有若干颗。
2. 两人轮流取子。每次从某一堆中取走至少 1 颗（可以取走整堆）。
3. 取走最后一颗石子的人获胜。

这看似随机的游戏其实有数学必胜策略—— 用异或（XOR）运算就能判断当前位置是必胜还是必败。

💡 提示：AI 使用完美的 XOR 策略。 如果当前 XOR ≠ 0，AI 会走到 XOR = 0 的位置（必胜位置）。 如果当前已经 XOR = 0，AI 只能破坏它，给你创造机会。

🟢 XOR = 0 → 安全位置（P-position），轮到谁谁输（如果对手不犯错）
🔴 XOR ≠ 0 → 危险位置（N-position），轮到谁谁赢（存在一步走到安全位置）

必胜位置图（P-position 地图）

第 3 章

在两堆尼姆游戏中，位置 (x,y) 是安全位置（🟢 P-position）当且仅当 x XOR y = 0，即 x = y。 图中绿色格子是安全位置——只要走到对角线上，对手就处于必败之地。

对于三堆及以上，安全位置的条件是：所有堆数的 XOR = 0。 例如 (1,2,3) 是安全位置，因为 1⊕2⊕3 = 0。

XOR 必胜策略详解

第 4 章

策略（Bouton 定理，1901）：

1. 计算所有堆石子数的 异或（XOR）和：S = a₁ ⊕ a₂ ⊕ ... ⊕ aₙ

2. 如果 S = 0，当前处于必败位置（P-position）——只能祈祷对手犯错。

3. 如果 S ≠ 0，存在必胜操作：找到一堆 aₖ，使得 aₖ ⊕ S < aₖ，将该堆减至 aₖ ⊕ S。

为什么要这么做？ 因为操作后新 XOR 和 = 0，把对手逼入必败位置。对手任何合法操作都会破坏 XOR = 0，你又可以恢复。如此循环，最终你取走最后一颗。

"尼姆游戏是组合博弈论的'果蝇'——简单到可以彻底分析，却包含了所有不偏博弈的核心结构。"— 组合博弈论的经典评价

博弈论的延伸

第 5 章
🧩
Sprague-Grundy 定理
任何不偏博弈（双方操作相同）都等价于某个大小的尼姆堆。Grundy 数就是该博弈的"尼姆值"。
♟️
围棋与象棋
有偏博弈（双方操作不同）不能用尼姆策略。围棋和象棋过于复杂，不存在已知的必胜公式——但理论上存在"超弱解"。
💻
密码学
XOR 的不可逆性在流密码（一次性密码本）中是理论上的完美加密。尼姆的 XOR 策略与密码学有共同的数学结构。
🧠
AI 博弈树搜索
Minimax 算法、Alpha-Beta 剪枝、蒙特卡洛树搜索——本质上都在寻找"必胜/必败位置"的泛化版本。AlphaGo 就是这个思想在围棋上的实现。

关键要点

1.XOR 是尼姆的数学核心。所有堆石子数的异或和为 0 ⟺ 安全位置（对手只要不犯错你就赢）。
2.P-position 和 N-position。P-position（Previous player wins）= XOR=0；N-position（Next player wins）= XOR≠0。
3.必胜操作总是存在。从XOR≠0的位置，一定存在某堆减少后使XOR=0。从XOR=0的位置，任何合法操作都会破坏平衡。
4.Sprague-Grundy 定理推广了尼姆。任何不偏博弈都可映射为某个 Grundy 数，组合博弈的必胜分析统一于尼姆框架。
5.有限的完美信息博弈 = 可计算解。任何有限的、双方轮流、信息完全的博弈，理论上都存在必胜/必平/必败的分类——只是有些太复杂无法实际计算。
