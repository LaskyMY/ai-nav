AI 游戏特效工作流

Claude Code + DeepSeek V4 Pro + MCP · 从零搭建到一键部署

🖥Claude Code → 🔄API Router → 🧠DeepSeek V4 → 🔌MCP → 🎮UE5/Unity

这是什么

一套让 AI 直接在游戏引擎中创建特效的完整工作流。通过 MCP 协议连接 Claude Code 和 Unreal Engine 5 / Unity，实现自然语言驱动粒子系统、材质、蓝图的自动生成。

免费开源 支持 UE5 / Unity 全程约 25 分钟 新手友好

5 步完成部署

1
安装 Claude Codenpm i -g @anthropic-ai/claude-code
2
接入 DeepSeek V4 Pro API4 行环境变量，价格低至 1/5
3
选择 MCP ServerUE5 用 UnrealClaudeMCP / Unity 用 GladeKit MCP
4
安装引擎插件 + 启动桥接复制插件到项目 → 编译 → 启动 Python 桥
5
用自然语言创建特效"帮我创建一个火焰拖尾 Niagara 系统..."
打开完整部署文档 · 所有命令可复制执行

能做什么

Niagara 粒子 一句话生成完整粒子系统
材质编辑 节点级材质图表创建
蓝图逻辑 自动生成蓝图节点连接
批量资产 重命名、组织、迁移
性能分析 扫描关卡、给优化建议
代码生成 C++ / C# 组件自动编写
